수량 조절

<%- include(`${_src}/views/components/form/input-quantity-stepper.ejs`, { type: 'horiz' }) %> 
  사용 방법
  
  1) 기본(가로형)
  <%- include(`${_src}/views/components/form/input-quantity-stepper.ejs`, {
    type: 'horiz',
    name: 'quantity',
    value: 1,
    min: 1,
    max: 9999,
    step: 1,
    unit: '개'
  }) %>
  
  2) 세로형
  <%- include(`${_src}/views/components/form/input-quantity-stepper.ejs`, {
    type: 'vertical',
    name: 'quantity2',
    value: 10,
    min: 1,
    max: 9999,
    step: 1,
    unit: '박스'
  }) %>
  
  3) 옵션 추가
  <%- include(`${_src}/views/components/form/input-quantity-stepper.ejs`, {
    type: 'horiz',
    variant: 'is-compact',
    name: 'quantity3',
    value: 1,
    min: 1,
    max: 5,
    step: 1,
    unit: 'EA',
    disabled: false,
    required: false
  }) %>
  
  파라미터
  - type: 'horiz' | 'vertical' (기본값: 'horiz')
  - variant: wrapper에 추가할 클래스
  - name: input name
  - value: 초기값
  - min: 최소값 (기본값: 1)
  - max: 최대값 (기본값: Infinity)
  - step: 증감 단위 (기본값: 1)
  - unit: 단위 텍스트 (미입력 시 숨김)
  - disabled: 비활성화 여부 (기본값: false)
  - required: 필수 여부 (기본값: false)

  • 기본(가로형)

    개

  • 기본(세로형)

    박스

  • 최소 1 개 , 최대 5 개

    EA

  • input 입력 disabled 상태

    EA
