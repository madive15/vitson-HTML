스와이퍼 타입 사용 방법

              components/pg/swiper-type.ejs

  - 사용:
    < % - include(`${_src}/views/components/pg/swiper-type.ejs`, {
      slides: ['Slide 1', 'Slide 2', 'Slide 3'],
      swiperType: 'card',
      desktop: 5,
      offsetBefore: 75,
      offsetAfter: 75,
      showNavs: true,
      containerClass: 'swiper-container'
    }) % >

  - 옵션:
    - slides (필수): 슬라이드 데이터 배열 (문자열/HTML)
    - swiperType: 스와이퍼 타입 ('card', 'list' 등, 기본값: 'card')
      + 신규 타입은 src/assets/scripts/ui/swiper.js 에서 추가
    - desktop: 데스크톱에서 보여줄 슬라이드 수 (기본값: 5)
    - offsetBefore: 오프셋 before 값 (기본값: 75, null이면 속성 제거)
    - offsetAfter: 오프셋 after 값 (기본값: 75, null이면 속성 제거)
    - showNavs: 네비게이션 버튼 표시 여부 (기본값: true)
    - containerClass: 컨테이너 클래스 (기본값: 'swiper-container')

  • 예시 1) 카드 타입

    <%- include(`${_src}/views/components/pg/swiper-type.ejs`, {
  slides: [
    '<div class="vits-card">카드 1</div>',
    '<div class="vits-card">카드 2</div>',
    '<div class="vits-card">카드 3</div>',
    '<div class="vits-card">카드 4</div>',
    '<div class="vits-card">카드 5</div>'
  ],
  swiperType: 'card',
  desktop: 3,
  offsetBefore: 75,
  offsetAfter: 75,
  showNavs: true
}) %>
