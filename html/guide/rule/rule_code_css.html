Publishing Conventions

코드규칙

CSS 기본 규칙

파일명세

  • * 파일에 따라서 불필요한 항목은 삭제 할 수 있다. (예 : animate.css)

코드 들여쓰기

  • 1탭 들여쓰기 하며, 1탭의 크기는 공백 4칸으로 설정한다.

들여쓰기를 하지 않는 경우

상위 블럭의 자식노드로 들어갈 경우 1level 까지는 가독성을 위해 1Tab 들여쓰기 가능

줄바꿈

  • 공통으로 사용되는 속성의 선택자간의 줄 바꿈은 허용하며 이 외의 줄바꿈은 허용하지 않는다.
  • 박스단위 구분을 위해 단 사이 빈줄 추가(최대 2줄을 넘지않는다)

주석/블럭구분

CSS 주석 표기

  • 컨테이너 및 메뉴 구분은 하기 내용과 같은 주석으로 표시한다.

상세 코드 규칙

작은 따옴표('') 사용 범위

폰트의 선언, filter, content에 작은 따옴표('')를 사용한다.

세미콜론(;)

마지막 선언된 속성에도 세미콜론(;)을 사용한다.

공백의 사용

선택자 간, 중괄호간 한칸의 공백을 사용한다, 중괄호 안쪽 좌우에는 공백은 사용하지 않는다.

선택자

공통 선택자

'*' 공통 선택자는 웹 페이지의 성능을 저하를 가져오기 때문에 가급적 사용하지 않는다.
