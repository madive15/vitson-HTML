영역구분

영역구분
구분 예약어 설명
카테고리 또는 최상위 영역 -wrapper 컨텐츠 단위별 감싸줄 때
모듈/컴포넌트 영역 -wrap 컨텐츠 단위별 감싸줄 때
지역 영역 -area 블록 컨텐츠 분류할 때
그룹 영역 -group 인라인 컨텐츠 분류할 때
안쪽 영역 -inner 안쪽 공간처리가 필요할 때
머리글 영역 -header 제목이 들어간다.
본문 영역 -body 내용이 들어간다.
바닥글 영역 -footer 바닥글이 들어간다.

레이아웃

레이아웃
구분 예약어 설명
그리드 기본 .grid .col 너비 100%
그리드 균등분할 .grid.flex .col flex 레이아웃 방식
그리드 균등분할 .grid.table .col table 레이아웃 방식
그리드 자식 지정칸수 .grid .col-[n].col n개만큼 column 병합
그리드 자신 지정칸수 .grid.cols-[n] .col n개만큼 column 분할
레이아웃 유연하게 .fluide 고정 또는 최소너비를 없앤다.
레이아웃 유연하지 않은 .unfluide 고정 또는 최소너비를 갖는다.
