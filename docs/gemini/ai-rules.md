# AI Response & Logging Rules

Gemini(AI)가 프로젝트 내에서 작업을 수행할 때 반드시 준수해야 하는 공통 규칙입니다.

## 1. 언어 제한 (Language Constraint)

-   Gemini의 모든 답변, 설명, 코드 주석(특별한 요청이 없는 한)은 **반드시 한국어(Korean)**로만 작성합니다.
-   전문 용어는 영어로 병기할 수 있으나, 전체적인 문장 구성 및 설명은 한국어로 작성하는 것을 원칙으로 합니다.

## 2. 로그 기록 (Logging)

-   코드 수정, 파일 생성, 구조 변경 등 모든 작업 후에는 반드시 로그를 남깁니다.
-   **저장 위치**: `docs/gemini/logs/` 디렉토리
-   **파일명 형식**: `yyyy-mm-dd.md` (예: `2026-03-28.md`)
-   **로그 내용**:
    -   작업 시간 및 작업자(Gemini)
    -   수정된 파일 목록
    -   변경 사항 요약 (Before / After)
    -   작업 결과 및 검증 내용

## 3. 답변 스타일 (Response Style)

-   사용자의 의도에 따라 간결하고 명확하게 답변합니다.
-   기술적인 결정이나 변경 사항에 대해서는 항상 논리적인 근거를 제시합니다.
