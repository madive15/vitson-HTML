# hygen
Hygen은 자바스크립트 기반의 코드 제너레이팅 툴

## api
[go](https://www.hygen.io/docs/quick-start)

## 페이지 생성 명령어
```bash
npx hygen pages with-prompt
```

## 초기화
```bash
npx hygen init self
```

## 기본 만들기
```bash
npx hygen generator new myTest
npx hygen myTest new 
```

## 대화창 만들기
```bash
npx hygen generator with-prompt myTest
npx hygen myTest with-prompt
```
